h1. Email2sms

!http://farm3.static.flickr.com/2545/3859077628_55156d5b3e_t.jpg(Avarteq GmbH)!:http://www.avarteq.de
Email2sms has been developed by "www.avarteq.de":http://www.avarteq.de.

Email2sms is a free and pure Ruby E-Mail to sms gateway including an extensible filterchain to filter and manipulate incoming emails before sending them as text messages.

h1. Requirements

In order to run Email2sms you will need the following.

h2. Required gems

* developergarden_sdk
* tmail

h2. Other requirements

* IMAP Mailbox
* DeveloperGarden Account with credits (developergarden.com)

h1. Usage

Usage...

h2. Configuration

Setup variables in

bc. config/email2sms.yml

h2. Start

bc. bin/email2smsd start

h2. Stop

bc. bin/email2smsd start

Watch output in

bc. log/email2sms_main.rb.output

and

bc.  log/email2sms_main.rb.log

for detailed application state information and errors.

h2. Send e-mail

In order to send an sms just drop an email to your email adress configured in the email2sms.yml file.
If you use the subject filter then you will need to have the password in your subject line:

bc. Subject:
mypass +49177 12344567
Message body:
Whatever

**That's it. Have fun!**
